# WEB SITE Section (yui540-style)



## Structure

- **Section** – CSS Grid with `row-gap`, vertical padding, white background.
- **Title** – Double “WEB SITE” heading with:
  - First line: dark background (#333), white text.
  - Second line: absolute overlay, white background, dark text (split/reveal look).
  - Clip-path reveal animation on load.
- **Cards** – Grid of project cards with image, hover scale, and bottom gradient caption.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech

- Next.js 14 (App Router)
- React 18
- CSS Modules
- Noto Sans JP (Google Fonts)
